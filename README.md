# Used stack
  React/React Hooks/TS/SCSS/BEM
# Description
  React app where you can add and remove todos, besides you can pick album from albums list and show its photo.
# Installing
 - Fork and clone this repository

 - Run npm install in your terminal

 - Run npm start
